ClipboardPanel 功能模块编排历史剪贴板和收藏剪贴板面板。

- `Sources/ClipboardPanel/ClipboardViewModel.swift` — 面板状态、采集、持久化和写回编排。
- `Sources/ClipboardPanel/ClipboardAppearancePreferences.swift` — 呼出面板外观偏好存储。
- `Sources/ClipboardPanel/ClipboardPanelView.swift` — 卡片式历史和收藏面板视图。
