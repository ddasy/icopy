ClipboardPanel 源码编排历史剪贴板和收藏剪贴板界面。

- `ClipboardViewModel.swift` — 面板状态、采集、持久化和写回编排。
- `ClipboardAppearancePreferences.swift` — 呼出面板外观偏好存储。
- `ClipboardPanelView.swift` — 卡片式历史和收藏面板视图。
