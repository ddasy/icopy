ICopyCore 源码定义剪贴板领域模型与业务规则。

- `ClipboardItem.swift` — 剪贴板条目模型。
- `ClipboardCollection.swift` — 历史和收藏集合规则。
- `StickyCardAppearance.swift` — 桌面卡片外观值类型(透明度/字体/颜色/强度),纯数据。
- `StickyCardItem.swift` — 桌面卡片领域模型、内容模式、锁定态、分区规则、剪贴板来源和翻译状态。
- `StickyCardCollection.swift` — 桌面卡片集合规则:新建、删除、排序和按 id 修改。
